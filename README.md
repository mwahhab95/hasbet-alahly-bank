# حاسبة البنك الأهلي

هذا مشروع Android بسيط لقراءة رسائل البنك الأهلي وحساب:
- إجمالي المصروف
- إجمالي المضاف للكارت
- الصافي

## ملاحظات
- التطبيق مخصص للاستخدام الشخصي.
- يعتمد على رسائل SMS الخاصة بالبنك الأهلي.
- يحتاج صلاحية READ_SMS.
- القواعد الحالية:
  - الرسائل التي تحتوي على "تم خصم" تعتبر مصروف.
  - الرسائل التي تحتوي على "تم إضافة" أو "تحويل لحظي" تعتبر إضافة للكارت.

## الملفات المهمة
- MainActivity.java
- activity_main.xml
- AndroidManifest.xml

## تحسينات مستقبلية
- دعم أكثر من صيغة رسالة
- إضافة اسم التاجر
